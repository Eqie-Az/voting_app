Cara menjalankan project voting_app:
1. Import database nya kedalam xampp, laragon, dll.
2. clone projectnya
3. jalankan websitenya sesuai dengan caranya, semisal laragon, jalankan melalui folder www.
4. Register/buat akun baru.
4. Login menggunakan akun yang baru saja di register.
5. melakukan voting.
6. Jika sudah maka user tidak bisa melakukan voting untuk yang ke 2xnya.
7. Jika ingin melakukan CRUD, login menggunakan email admin, tertera di database.

Deskripsi Project:
Voting_app adalah aplikasi web berbasis PHP Native dengan arsitektur Model-View-Controller (MVC) yang dirancang untuk mempermudah proses pemilihan kandidat secara digital.
Aplikasi ini memungkinkan pengguna untuk melakukan voting secara online dengan sistem yang sederhana namun aman, serta memberikan kontrol penuh kepada admin dalam mengelola data kandidat dan hasil voting
